# HL7 v3 Spec

HL7 v3 classes generated from XSDs and IHE web services generated from their WSDLs.

## Usage

Add a dependency to your project
```xml
<dependencies>
    <groupId>com.parsek.vitaly</groupId>
    <artifactId>integration-hl7v3-spec</artifactId>
    <version>1.0.3</version>
</dependencies>
```
